Task: Task Manager App

Objective

Develop a Task Manager App using React and React Hooks to manage a to-do list with advanced functionality, while ensuring performance optimization.

Functional Requirements

User Management
  • Add new tasks
  • Mark tasks as completed
  • Delete tasks
Task Management
  • Unique ID for each task (auto-generated)
  • Task Text (required)
  • Completed Status (boolean)
  • Store tasks in React Context using useReducer
  • Persist tasks in local storage
Timer
  • Auto-start on app launch
  • Increment every second
  • Reset timer functionality
Performance Optimizations
  • useMemo for completed task count calculation
  • useCallback to prevent unnecessary re-renders of functions
  • React.memo to optimize re-renders of components

Technical Requirements

React Hooks Usage
  • useState: Manage local state (timer, input field)
  • useEffect: Automatically increment timer
  • useContext: Store tasks globally
  • useReducer: Handle task actions efficiently
  • useRef: Access input field & timer reference
  • useMemo: Optimize completed task count calculation
  • useCallback: Prevent unnecessary function re-creations
  • useImperativeHandle: Expose a resetTimer method from the Timer component

UI

Requirements
  • Text input field for tasks
  • "Add Task" button
  • List of tasks (with checkbox and remove button)
  • Completed tasks counter
  • Timer display
  • "Reset Timer" button

Implementation

Task State Management (useContext + useReducer)
  • Use TaskContext provider for global state
  • useReducer for handling ADD_TASK, TOGGLE_TASK, REMOVE_TASK
Timer (useRef, useEffect, useImperativeHandle)
  • Store timer in useRef
  • Start timer on component mount (useEffect)
  • Allow resetting from parent (useImperativeHandle)
Performance Optimizations (useMemo, useCallback)
  • Use useMemo for completed task count calculation
  • Wrap addTask function with useCallback

Deliverables

  • Fully working Task Manager app
  • Correctly implemented React hooks
  • Optimized performance using memoization
  • Clean, maintainable code
