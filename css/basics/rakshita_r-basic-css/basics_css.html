APTY CONTENT BOX

WORKFLOWS

Apty Workflows, which handholds users to complete a task in your application, eases onboarding, and leads users through challenging duties while ensuring that no crucial steps are skipped.

TOOLTIPS

Tooltips serve as a way to provide users with additional information about the elements within your application. Each Tooltip is represented by an 'i' icon located next to the respective element.

LAUNCHERS

Launchers are actionable icons or buttons that trigger predefined workflows or important information to your end users needing immediate attention. You can launch Workflows, videos, links, images, and text using Launchers.
