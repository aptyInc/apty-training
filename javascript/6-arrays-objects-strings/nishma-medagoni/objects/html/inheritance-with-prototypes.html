Object Inheritance Using Prototypes

In JavaScript, inheritance can be implemented using prototypes. A prototype is a property of an object that points to another object, allowing one object to inherit properties and methods from another. In this example, we have two objects: Person and Employee.

- The Person object has a name property and a greet method.

- The Employee object inherits from the Person object using Object.create(person) and adds a new job property.

- When we call employee.greet(), it uses the method from the Person object but with the added information from the Employee object, resulting in a personalized greeting.
