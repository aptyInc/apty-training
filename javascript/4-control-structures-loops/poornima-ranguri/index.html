Conditions Assignments

Grading System

Create a program to take marks as input from the user (e.g., using a number input) and print the grade (A, B, C, F) back to the HTML page based on conditions: 90+ => Grade A, 75-89 => Grade B, 50-74 => Grade C, Below 50 => Grade F.

Leap Year Checker

Create a program that takes a year as input from the user and checks if the year is a leap year. Print the result ("Leap Year" or "Not a Leap Year") back to the HTML page.

Simple Calculator

Clone the calculator app you had built in previous assignment and create calculate(operator) function to get the results. T his function should handle all operations, with the operator passed as a parameter for each button.

Find the largest number

Take three numbers as input from the user and find the largest number among them. Print the result back to the HTML page.

Odd or Even

Take a number as input from the user and determine whether it is odd or even. Print the result back to the HTML page.

Simple Intrest Calculator

Take inputs for principal amount, rate of interest, and time from the user, and calculate the simple interest. Print the result back to the HTML page.

Loops Assignments

Generate Multiplication Table

Take a number as input from the user and generate its multiplication table using a for loop. Display the table in an HTML page.

Sum of Numbers

Use a while loop to calculate the sum of all numbers from 1 to a user-specified number. Display the result in a paragraph below the input.

Factorial Representation

Take a number as input and calculate its factorial using a do-while loop. Show the calculation steps as well as the final result.

Array Sum and Average

Take an array of numbers as input from the user (comma-separated) and use a for-of loop to calculate the sum and average. Display both results on the webpage.

Character Frequency

Prompt the user for a string and use a for-in loop to count how many times each character appears in the string. Display the results on the webpage.

Pyramid Pattern

Use a for loop to create a pyramid of stars(*) based on the number of rows input by the user. Display the pattern using preformatted text

Print Your Name

Create two dimensional array with all required letters. Use nested for-of to read and print values in console and attach results screenshot in html page
