HTML Training Topics and Assignment1

Topics

  • Structure of an HTML document
  • Tags, elements, attributes
  • Basic tags: <html>, <head>, <body>, <title>, <h1> to <h6>, <p>, <a>
  • Ordered and unordered lists (<ol>, <ul>, <li>)
  • Nesting lists
  • Adding images (<img>) and alt text
  • Creating tables (<table>, <tr>, <td>, <th>, <caption>)
  • Form basics: <form>, <input>, <label>, <button>

Assignments

  • Create a basic webpage with a title, headings, paragraphs, and a link to another page.
  • Design a webpage with a list of your favorite movies and an image for each.
  • Build a webpage with a table for a class timetable and a simple login form.
