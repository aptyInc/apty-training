HTML Training Topics and Assignment2

Topics

  • Semantic elements: <header>, <footer>, <article>, <section>, <nav>, <aside>
  • Form validation attributes: required, pattern, min, max
  • Advanced inputs: <email>, <tel>, <date>, <number>
  • Internal and external links
  • Linking to specific sections using anchors (<a href="#section">)
  • Embedding videos and audio (<video>, <audio>)
  • Customizing media controls

Assignments

  • Create a webpage using semantic elements to structure content like a blog.
  • Create a registration form with all input types and proper validations.
  • Build a multi-page website with working navigation links.
  • Add a video and an audio clip to a webpage with captions and descriptions.
